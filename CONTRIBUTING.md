# Contributing

Thanks for your interest in improving the Crew Performance Scorecard Dashboard!

- Fork the repository and create a new branch for your changes.
- Keep pull requests focused on a single, well-scoped improvement.
- Include clear descriptions of the changes and any manual testing performed.
- Follow the existing HTML, CSS, and JavaScript style patterns used in the project.

By contributing, you agree that your submissions will be licensed under the same license as the project.
